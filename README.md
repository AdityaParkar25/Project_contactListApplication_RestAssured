Telecom API Project

1. Project Name:-
API testing for Contact List Application.

2. Objective:-
To perform comprehensive testing of the APIs provided by the https://thinking-tester-contact-list.herokuapp.com/ Contact List application.

3. Technology Stack:-
•	Postman: for manual API testing and assertions.
•	Chai: for API assertions in JavaScript.
•	RestAssured: for API testing in Java.
•	Git: version control for RestAssured code.

4. Reporting:-
TestNG default report
